Welcome to my DevOps practice and professional projects repository. This repository includes a variety of projects aimed at helping developers and DevOps practitioners gain practical experience with DevOps practices. The projects include CI/CD pipelines, IaC scripts, testing frameworks, monitoring, logging tools, containerization & orchestration tools. Each project contains code, documentation,, instructions for deployment. Feel free to explore the repository and these projects to improve your DevOps skills or if you wish to contribute, would love that!
